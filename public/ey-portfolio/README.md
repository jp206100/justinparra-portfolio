# EY Portfolio

Static case-study page. Drop the `ey-portfolio/` folder into your `justinparra.com` repo (at the root, or inside `public/` if your main site has a build step) — it will be served at `justinparra.com/ey-portfolio/`.

All asset paths are relative, so no config needed.

## Local preview

```bash
cd ey-portfolio
npx serve .
```
